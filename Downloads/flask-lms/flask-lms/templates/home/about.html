About us

Flask LMS is a lightweight learning platform built to demonstrate modern full-stack Python development.

Features include role-based auth, course management, PDF & YouTube materials, quizzes, progress tracking, plus AI-powered tutoring and quiz generation.

Tech stack

  • Python Flask + Jinja2
  • SQLite via SQLAlchemy ORM
  • Flask-Login for sessions, Flask-WTF for forms/CSRF
  • Bootstrap 5 for a responsive UI
  • OpenAI-compatible API for AI tutor & quiz generation
